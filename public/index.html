Docs for API

Welcome to the docs of we API. This page give all informations for integrate with we services.
URL Base: https://apiaubaro.vercel.app/v1

Endpoints

GET /AdUsers

Return a list of all users with authetication to make a docs update. The users with the permission are Admin's

Example of Request

GET https://apiaubaro.vercel.app/v1/users/AdUsers

POST /newUser

Create a new user for the sistem.

Parameters

Name Type Obrigatório Descrision
name string yes Name of user (20 caracteres in max)
email string yes E-mail of user (30 caracteres in max)
Password string yes Password for acess (min 8 and max 16 caracteres )
isAdmin string no if one user is type Admin, default value is "no".

Exempla of Request

POST https://apiaubaro.vercel.app/v1/users/newUser
                Content-Type: application/json
                {
                    "name": "Carlos Oliveira",
                    "email": "carlos@exemplo.com",
                    "password": "senhaSegura123",
                    "isAdmin": 'no'
                }

Exempla of Respost

{
                "id": 3,
                "name": "Carlos Oliveira",
                "email": "carlos@exemplo.com",
                "date_create": "2023-05-07T09:15:00Z"
            }
GET /docs

Return a list with all docs

POST /docs/newDocs

Insert a new doc

Códigos de Status

Código Descrição
200 OK - Requisição bem-sucedida
201 Created - Recurso criado com sucesso
400 Bad Request - Parâmetros inválidos
401 Unauthorized - Autenticação necessária
404 Not Found - Recurso não encontrado
500 Internal Server Error - Erro no servidor

Limites e Cotas

  • Limite de requisições: 100 requisições por minuto por token
  • Tamanho máximo de payload: 1MB
  • Timeout: 30 segundos por requisição
